# smartrade-adk
A smart trade agent.

## 项目概述

本项目基于[Google ADK](https://github.com/google/adk-python)和[AG-UI](https://github.com/ag-ui-protocol/ag-ui)构建的A股投资智能体。

## 技术栈

- 后端：Python 3.8 + FastAPI + Google ADK + AG-UI
- 前端：Next.js + Shadcn/ui + Tailwind CSS + Lucide-react
- 数据库与文件存储：Supabase
